# Mats Presentation

En enkel statisk webbplats som presenterar min karriär — bilder och en 90-tals rap som beskriver resan.

## Innehåll
- `Index.html` — huvudsidan med systemtema, bildgalleri och ljudspelare.
- `pics/` — bildfiler som används i galleriet.
- `music/mellan_betong_och_gr_nt.mp3` — låten som spelar upp under presentationen.

## Snabbstart

### Öppna lokalt
1. Öppna filen `Index.html` i din webbläsare (dra och släpp i ett fönster eller högerklick → Öppna med).
2. Klicka på play-knappen i sektionen **MUSIC** för att spela upp låten.
3. Klicka på valfri bild i galleriet för att förstora (lightbox).

### Öppna via GitHub Pages
Webbplatsen kan publiceras via GitHub Pages. Om aktivt publicerad kommer den att vara tillgänglig på:

https://matsromblad.github.io/mats-presentation/

Om du inte ser sidan direkt — vänta 1–5 minuter efter att Pages aktiverats.

## Kompatibilitet
- HEIC-bilder kan behöva konverteras för att fungera i alla webbläsare. Jag kan konvertera dessa åt dig om du vill.
- Moderna webbläsare kräver oftast att användaren interagerar innan ljud spelas upp (autoplay kan blockeras).
